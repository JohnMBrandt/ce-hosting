About Restoration Mapper

Overview

Restoration Mapper is an online tool to create wall-to-wall maps of tree presence using open source artificial intelligence and open source satellite imagery. The Restoration Mapper approach enables monitoring stakeholders to:

  • Rapidly assess tree density in non-forested landscapes
  • Establish wall-to-wall baseline data
  • Measure yearly change in tree density without conducting follow-up mapathons
  • Generate maps relevant to land use planning
  • Identify agroforestry, riparian buffer zones, and crop buffer zones
  • Generate GeoTIFFs for further spatial analysis or combination with other datasets

Usage

Restoration mapper is open source. All code and data is available for use and is hosted on GitHub. While one can request data without any programming through this form, the original code can be altered and adapted by cloning the Docker container.

Methodology

Restoration mapper uses open source artificial intelligence, powered by TensorFlow. The approach is specifically designed to allow for the identification of small and scattered trees in heterogeneous and mosaic landscape. The preprint of the methodology is available here.
